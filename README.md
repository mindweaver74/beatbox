# 🎤 비트박스 마스터: Drop the Beat!

> 우주에서 쏟아지는 비트를 격파하라! B·S·K 타이밍 완벽 재현!

**▶ 지금 바로 플레이** → [https://mindweaver74.github.io/beatbox](https://mindweaver74.github.io/beatbox)

---

## 🎮 게임 방법

1. 화면 상단에서 **B / S / K** 노트가 아래로 내려옵니다
2. 노트가 하단 **판정선**에 닿는 순간 해당 키를 누르세요!
3. 타이밍이 정확할수록 높은 점수 — **PERFECT** 를 노려라!
4. 10콤보 달성 시 🔥 **싸이퍼 피버타임** 발동!

---

## 🕹️ 조작법

| 환경 | B 레인 | S 레인 | K 레인 |
|------|--------|--------|--------|
| PC (기본) | `B` 키 | `S` 키 | `K` 키 |
| PC (왼손) | `A` 키 | `S` 키 | `D` 키 |
| 모바일 | `B` 버튼 | `S` 버튼 | `K` 버튼 |

---

## ⚡ 판정 기준

| 판정 | 타이밍 | 점수 | 색상 |
|------|--------|------|------|
| ✨ PERFECT | ±45ms | 300점 | 금색 |
| 💙 GREAT | ±85ms | 200점 | 파랑 |
| 💚 GOOD | ±130ms | 100점 | 초록 |
| ❌ MISS | 초과 | 0점 | 빨강 |

> 콤보가 쌓일수록 점수 배율 증가! (10콤보마다 +0.5배)

---

## 🔥 싸이퍼 피버타임

**10콤보** 달성 시 5초간 발동!

- 노트가 **황금색**으로 변환
- B·S·K **어느 키든** 연타해도 점수 인정
- 점수 **2배** 적용
- 배경 이퀄라이저 광란 애니메이션
- 관객 캐릭터들 미친 듯이 춤!

---

## 🥁 비트박스 사운드

노트를 맞출 때마다 찰진 비트박스 사운드가 터져요!

| 레인 | 사운드 | 느낌 |
|------|--------|------|
| B | 킥드럼 | 쿵! |
| S | 스네어 | 착! |
| K | 하이햇 | 칫! |

> 타이밍을 잘 맞출수록 하나의 완성된 비트박스 루프가 탄생해요!

## 🌌 비주얼 특징

- **우주 배경** — 반짝이는 별빛 파티클
- **네온 레인** — B(파랑) / S(보라) / K(주황) 3색
- **이퀄라이저** — 하단에 실시간 춤추는 EQ 바
- **피버 글로우** — 황금 네온 전체 화면 발광
- **접근 펄스** — 판정선 근처서 노트 외곽 빛남
